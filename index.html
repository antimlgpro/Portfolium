I make websites and games

I have been making games since 2014. During that time I learned a lot about both programming in general and game design. Most games made while learning never became more than small prototypes, but i never made something that did not teach me new stuff.

Around the same time I started making websites. I quickly taught myself HTML, CSS and Javascript by simply doing projects that i thought i could make. Nothing i know comes from teachers in school, but instead from Google and Youtube.
When I felt I had mastered Vanilla web design i started learning Javascript Frameworks like React and Svelte. Im not yet an expert in these frameworks but im always learning new stuff.

During my late teen years I learned OpenGL and Rust by making my own simple game engine, in its final state it could not do much but taught me a lot about low-level 3D rendering.

I have a passion for programming that lets me make almost anything I feel inspired to make.

Simple low-level game engine
